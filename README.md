# Bots template for play in islands Game on GrandGames Arena

- Install dependencies `npm i`
- Create your AI based `src/ai/GameAI.ts`. Example in `src/ai/RandomMove/RandomMoveAI.ts`
- To test answers of your AI change `/src/test/testPos.ts` and run `npm run test`.
- Helper functions to simplify your AI development are in `src/utils/` folder.

## When you are ready to play

- Get token from GrandGames Arena and put it in `.env` file
- `npm run build` to build the project
- `npm run start` to start the bot

See for more info about sdk: https://github.com/MemoS77/GrandGames-Arena-Bots-SDK
